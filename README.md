# 점프! 점프! 🐸

두들 점프 스타일의 무한 점프 웹 게임입니다. 순수 HTML5 Canvas + JavaScript로 만들어져 의존성이 없고, PWA를 지원해 폰 홈 화면에 앱처럼 설치할 수 있습니다.

## 플레이 방법

| 조작 | 모바일 | PC |
|------|--------|-----|
| 좌우 이동 | 화면 좌/우 터치 **또는** 기기 기울이기 (시작 화면에서 선택) | ← → 방향키 |
| 발사 | 화면 상단(위쪽 25%) 탭 | 스페이스바 |
| 일시정지 | 우상단 ❚❚ 버튼 | ESC |

## 게임 요소

- 🟢 **초록 발판**: 일반 발판
- 🔵 **파란 발판**: 좌우로 움직임
- ⚪ **흰 발판**: 한 번 밟으면 사라짐
- 🟤 **갈색 발판**: 밟으면 부서짐 (튕기지 못함)
- 🔴 **스프링**: 더 높이 점프
- 🟠 **제트팩**: 잠시 동안 빠르게 상승
- 👾 **몬스터**: 위에서 밟거나 총알로 처치. 옆이나 아래에서 부딪히면 게임 오버

처음 플레이하면 발판 종류를 설명하는 "게임 방법" 화면이 나오고, 시작 시 3초 카운트다운 후 게임이 시작됩니다. 처음에는 발판이 매우 촘촘하고(간격 30~44px) 몬스터도 없어 쉽게 시작하고, 높이 올라갈수록 발판 간격이 넓어지고 **발판 자체도 작아지며**, 특수 발판·몬스터가 늘어납니다. 일부 발판은 늘었다 줄었다 크기가 변합니다. 배경은 **지상(풀밭) → 하늘 → 성층권 → 우주**로 변하고, 가끔 비나 눈이 내립니다.

## 미션 시스템

플레이 중 화면 상단에 미션이 나타납니다. 완수하면 **보너스 타임** — 제트팩을 타고 무적 상태로 한참 상승합니다! 완수 후 잠시 뒤 새 미션이 등장하며, 높이 올라갈수록 다양한 미션이 풀립니다.

| 미션 | 등장 조건 |
|------|-----------|
| 새 발판 10개 연속 밟기 | 처음부터 |
| 코인 10개 모으기 | 처음부터 |
| 스프링 2번 타기 | 처음부터 |
| 총알 5발 쏘기 | 처음부터 |
| 15초 안에 700점 오르기 (실패 시 자동 재도전) | 처음부터 |
| 파란 발판 3번 밟기 | 1,000점부터 |
| 흰 발판 3번 밟기 | 1,000점부터 |
| 초록·파랑·흰 발판 모두 밟기 | 1,500점부터 |
| 몬스터 1마리 처치 | 1,800점부터 |

## 코인과 상점

- 🪙 **코인**: 하늘에서 떨어집니다. 모은 코인은 자동 저장되어 다음 판에도 유지됩니다.
- ⭐ **별**: 가끔 코인 대신 별이 떨어집니다. **10개 모으면 스타 파워** — 짧은 시간 무적 상태로 하늘을 날아오릅니다!
- 🛒 **상점** (시작 화면):
  | 아이템 | 가격 | 효과 |
  |--------|------|------|
  | ❤️ 생명 | 🪙 150 | 죽어도 한 번 부활 (화면 아래에서 튀어오르며 잠시 무적, 최대 3개 보유) |
  | 🚀 로켓 출발 | 🪙 50 | 게임 시작 시 제트팩으로 발사 (게임 시작 시 1개 자동 사용) |
  | 🧲 코인 자석 | 🪙 75 | 한 판 동안 주변 코인을 끌어당김 (게임 시작 시 1개 자동 사용) |

## 캐릭터 교체 방법

`assets/character/` 폴더에 아래 PNG 파일을 넣으면 자동으로 적용됩니다 (투명 배경 권장, 정사각형 비율 권장):

| 파일명 | 설명 | 필수 여부 |
|--------|------|-----------|
| `jump-left.png` | 기본/하강 모습 | ✅ 필수 |
| `fly-left.png` | 점프로 상승 중일 때 모습 | 선택 |
| `jump-right.png` | 오른쪽을 보는 모습 | 선택 (없으면 좌우반전) |
| `shoot.png` | 위로 발사할 때 모습 | 선택 |

현재 기본 캐릭터는 흰색 말티즈 강아지입니다. 위 파일을 교체하면 다른 캐릭터로 바뀝니다.

## 로컬에서 실행

```bash
cd game
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

## 배포

`main` 브랜치에 push하면 GitHub Actions(`.github/workflows/deploy.yml`)가
자동으로 `gh-pages` 브랜치에 배포합니다.

배포 주소: **https://jiuk96.github.io/jump-jump/**

> 최초 1회, 저장소 Settings → Pages 에서 Source가 `gh-pages` 브랜치로 잡혀 있는지 확인이 필요할 수 있습니다.

## 폰에 앱처럼 설치 (PWA)

- **iPhone (Safari)**: 공유 버튼 → "홈 화면에 추가"
- **Android (Chrome)**: 메뉴 → "홈 화면에 추가" 또는 설치 배너

설치하면 전체 화면으로 실행되고, 오프라인에서도 플레이할 수 있습니다.
