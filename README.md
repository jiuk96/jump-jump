# 점프! 점프! 🐸

두들 점프 스타일의 무한 점프 웹 게임입니다. 순수 HTML5 Canvas + JavaScript로 만들어져 의존성이 없고, PWA를 지원해 폰 홈 화면에 앱처럼 설치할 수 있습니다.

## 플레이 방법

| 조작 | 모바일 | PC |
|------|--------|-----|
| 좌우 이동 | 화면 좌/우 터치 **또는** 기기 기울이기 (시작 화면에서 선택) | ← → 방향키 |
| 발사 | 오른쪽 아래 🔫 버튼 | 스페이스바 |
| 일시정지 | 우상단 ❚❚ 버튼 | ESC |

## 게임 요소

- 🟢 **초록 발판**: 일반 발판
- 🔵 **파란 발판**: 좌우로 움직임
- ⚪ **흰 발판**: 한 번 밟으면 사라짐
- 🟤 **갈색 발판**: 밟으면 부서짐 (튕기지 못함)
- 🔴 **스프링**: 더 높이 점프
- 🟠 **제트팩**: 잠시 동안 빠르게 상승
- 👾 **몬스터**: 위에서 밟거나 총알로 처치. 옆이나 아래에서 부딪히면 게임 오버

처음 플레이하면 발판 종류를 설명하는 "게임 방법" 화면이 나오고, 시작 시 3초 카운트다운 후 게임이 시작됩니다. 처음에는 발판이 매우 촘촘하고(간격 30~44px) 몬스터도 없어 쉽게 시작하고, 높이 올라갈수록 발판 간격이 넓어지고 **발판 자체도 작아지며**, 특수 발판·몬스터가 늘어납니다. 일부 발판은 늘었다 줄었다 크기가 변합니다. 배경은 **지상(풀밭) → 하늘 → 성층권 → 우주**로 변하고, 가끔 비나 눈이 내립니다.

## 미션 시스템

플레이 중 화면 상단에 미션이 나타납니다. 완수하면 **보너스 타임** — 제트팩을 타고 무적 상태로 한참 상승합니다! 완수 후 잠시 뒤 새 미션이 등장하며, 높이 올라갈수록 다양한 미션이 풀립니다.

| 미션 | 등장 조건 |
|------|-----------|
| 새 발판 10개 연속 밟기 | 처음부터 |
| 코인 10개 모으기 | 처음부터 |
| 스프링 2번 타기 | 처음부터 |
| 총알 5발 쏘기 | 처음부터 |
| 15초 안에 700점 오르기 (실패 시 자동 재도전) | 처음부터 |
| 파란 발판 3번 밟기 | 1,000점부터 |
| 흰 발판 3번 밟기 | 1,000점부터 |
| 초록·파랑·흰 발판 모두 밟기 | 1,500점부터 |
| 몬스터 1마리 처치 | 1,800점부터 |

## 콤보 & 연출

- **콤보**: 새 발판을 연속으로 밟으면 콤보가 쌓입니다 (같은 발판을 다시 밟으면 리셋). 5콤보부터 보너스 점수, 10콤보마다 +3🪙
- **아슬아슬**: 발판 끝에 겨우 착지하면 표시, 죽거나 부딪히면 화면 흔들림
- **고스트**: 최고 기록 판의 궤적이 반투명 둥이(👻)로 함께 올라갑니다
- **최고 기록 깃발**: 지난 최고 기록 높이에 🚩 점선이 표시됩니다

## 상승 구간 재미 요소

- ❄️ **얼음 발판**: 밟으면 잠시 미끄러워 잘 안 멈춥니다. 눈이 올 때 훨씬 자주 등장
- 💨 **바람 지대** (2,500점~): 경고 후 한동안 옆바람이 캐릭터를 밀어냅니다
- 🌌 **무중력 구간** (13,500점~): 우주에선 중력이 절반 — 점프가 둥실둥실 높아집니다
- 🎯 **대포** (1,200점~): 떨어져서 들어가면 포신이 좌우로 왕복 조준 — 탭 타이밍에 맞춰 발사! (2초 내 입력 없으면 자동 발사)
- 🌈 **무지개 코인**: 코인 5개 가치, 색이 변하며 반짝입니다

## 🔥 불길

게임 시작 직후부터 **아래에서 불길이 계속 올라옵니다** — 같은 곳에 머물 수 없습니다! 높이 올라갈수록 불길도 빨라지고, 멀리 도망가도 화면 아래까지는 따라붙습니다. 닿으면 사망 (부활 시 불길이 잠시 밀려남). 강화의 "내열 코팅"으로 속도를 늦출 수 있습니다. 보스전 중에는 멈춥니다.

## 미니보스

6,000점마다 보스가 등장하며, **티어가 오를수록 보스의 모습과 공격 패턴이 진화**합니다:

| 티어 | 보스 | 패턴 |
|---|---|---|
| 1 (6천) | 👹 젤리킹 (보라·뿔) | 조준탄 · 3갈래 · 양옆 낙하 |
| 2 (1.2만) | 🔥 화염 대왕 (빨강·불꽃 갈기) | + 벽 탄막 (틈으로 피하기!) |
| 3 (1.8만) | ❄️ 얼음 마왕 (파랑·얼음 결정) | + 지그재그탄(하늘색), 5갈래 |
| 4+ (2.4만~) | 👑 암흑 제왕 (흑자색·황금 왕관·오라) | + 유도탄(보라), 조준 2연발 |

보스가 등장하면 **아레나 전투**가 시작됩니다 — 카메라가 멈추고 일반 발판이 잠시 사라지며 화면 전체를 덮는 일자 땅이 생기고, **점프가 멈추고 발판 위에 서서** 좌우 이동으로 보스의 3가지 공격 패턴(조준탄·3갈래 부채꼴·양옆 낙하탄)을 피하며 🔫 버튼으로 반격합니다. 발판에서 떨어져도 죽지 않고 바닥 발판으로 복귀합니다 (투사체만 조심!). 격파하면 코인 + 300점 보너스! (25초 버티면 보스가 떠나고 진행 재개)

## 달 착륙 엔딩 🌕

**30,000점에 도달하면 달에 착륙**하며 게임을 클리어합니다! (4번째 보스 암흑 제왕을 넘어서야 합니다) 둥이가 깃발을 꽂는 엔딩 연출과 함께 "달 정복자" 도전과제(+500🪙)를 얻습니다. 올라가는 길에는 구간 이정표(구름·성층권·우주·달)가 표시됩니다.

## 가위바위보 부활

생명·고양이 목숨을 다 쓰고 죽으면 마지막 기회로 **몬스터와 가위바위보**를 합니다 (판당 1회). 이기면 한 목숨 더, 비기면 다시, 지면 게임 오버!

## 온라인 랭킹

- 게임오버 시 점수가 자동으로 랭킹 서버(Supabase)에 등록됩니다 (최초 1회 닉네임 입력, 100점 미만은 제외). 사용한 캐릭터도 함께 기록됩니다
- 시작 화면 🏅 랭킹은 **TOP 10** — 1·2·3등은 🏆 **명예의 전당 시상대**에 달성 캐릭터가 올라섭니다 (1등이 가운데 제일 높이, 👑 왕관 애니메이션). 4~10등은 리스트로 표시되고 내 기록은 강조됩니다
- 같은 닉네임은 최고 점수만 표시됩니다
- (운영 메모) 시상대에 캐릭터가 나오려면 Supabase SQL Editor에서 1회 실행: `alter table public.scores add column charid text;` — 컬럼이 없어도 랭킹은 정상 동작하며 기본 캐릭터(둥이)로 표시됩니다

## 로컬에서 실행

```bash
cd game
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

## 배포

`main` 브랜치에 push하면 GitHub Actions(`.github/workflows/deploy.yml`)가
자동으로 `gh-pages` 브랜치에 배포합니다.

배포 주소: **https://jiuk96.github.io/jump-jump/**

> 최초 1회, 저장소 Settings → Pages 에서 Source가 `gh-pages` 브랜치로 잡혀 있는지 확인이 필요할 수 있습니다.

## 폰에 앱처럼 설치 (PWA)

- **iPhone (Safari)**: 공유 버튼 → "홈 화면에 추가"
- **Android (Chrome)**: 메뉴 → "홈 화면에 추가" 또는 설치 배너

설치하면 전체 화면으로 실행되고, 오프라인에서도 플레이할 수 있습니다.
